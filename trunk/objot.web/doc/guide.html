面向服务的Ajax

定义服务对象Service Object

借助于Objot轻量级对象序列化反序列化机制，View和Service之间的数据交互相当简洁而高效。 可以传递的数据类型包括：

JavascriptJavaPython
null undefinednullNone
booleanbooleanbool
numberint long(部分) float double Integer Long(部分) Float Doubleint long(部分) float
stringStringunicode str(UTF-8)
Arrayboolean[] int[] long[] <T>[] List<?>list tuple
Object(非Function String Number Boolean Array)Object Map<String,Object>object(非instance) dict
对象之间可以任意引用，整个对象图都可以被传递，同时也可以定义规则，只传递对象图的局部（子图）。 Objot还支持单继承类，服务端和浏览端可以使用不同的类名，并可以对应。 对象的属性名两端也可不同，可以对应。

传递的对象图必须以list或者object为起点，传递的对象被称为服务对象Service Object。
